# linux-firewall-lab
Implementación de un cortafuegos Linux seguro mediante iptables y nftables, incluyendo filtrado de paquetes, seguimiento de conexiones, registro de eventos y control de acceso SSH.

## Skills demonstrated

- Linux System Administration
- Firewall Design
- Packet Filtering
- Network Security
- Connection Tracking (conntrack)
- SSH Hardening
- Traffic Logging
- nftables Migration
- Security Troubleshooting
- Linux Networking
